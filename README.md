# 🌦️ Weather Data Analysis and Prediction

## 📌 Overview
Weather Data Analysis and Prediction is a Machine Learning project that analyzes historical weather data to identify trends and patterns, performs data preprocessing and visualization, and predicts future weather conditions using predictive models.

This project demonstrates the complete data science workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, model building, evaluation, and prediction.

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Weather Trend Visualization
- Feature Engineering
- Machine Learning Model Training
- Weather Prediction
- Model Performance Evaluation

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- XGBoost
- Jupyter Notebook

## 📈 Machine Learning

The project uses machine learning algorithms to predict weather conditions based on historical weather data.

Evaluation metrics include:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Weather-Data-Analysis-and-Prediction.git
```

2. Navigate to the project folder

```bash
cd Weather-Data-Analysis-and-Prediction
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open:

```
Weather_Data_Analysis_and_Prediction.ipynb
```

and run all cells.
